# Student Management System

A web-based **Student Management System** built using **Flask** and **SQLite** that provides a simple and efficient way to manage student records. The application includes secure user authentication, a user-friendly dashboard, and complete CRUD (Create, Read, Update, Delete) functionality for student data management.

## Features

* User Registration and Login System
* Secure Session-Based Authentication
* Interactive Dashboard
* Add New Student Records
* Update Existing Student Information
* Delete Student Records
* Search Students by:

  * Student ID
  * Name
  * Roll Number
  * Department
  * Course
* SQLite Database Integration
* Responsive and Easy-to-Use Interface

## Technologies Used

* Python
* Flask
* SQLite
* HTML5
* CSS3
* Jinja2 Templates

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```bash
   cd student-management-system
   ```

3. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the application:

   ```bash
   python app.py
   ```

5. Open your browser and visit:

   ```
   http://127.0.0.1:5000
   ```
